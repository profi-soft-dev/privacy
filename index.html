Privacy Policy for ContentAuto

This Privacy Policy describes how the "ContentAuto" application handles data.

1. Data Access

Our application uses the Pinterest API solely to automate the uploading of original design content and infographics created by the user.

2. Personal Information

We do not collect, store, or share any personal identifiable information, login credentials, or contact details of Pinterest users.

3. Data Security

All interactions with the Pinterest API are performed through secure integration. No data is transferred to third parties.

4. Contact

If you have any questions, please contact the developer via the official contact information.
